About HTML

HTML consists of open tags, closed tags, attributes and contents. It is used to specify the content of webpages.

You can learn more about HTML by visiting the W3Schools site's tutorial about HTML.

About CSS

CSS is used for formatting webpages: fonts, colors, layout, etc. There're different ways to specify CSS:

  1. Directly in an element
  2. In a script element in the head
  3. In an external CSS file
